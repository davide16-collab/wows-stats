How it works

What the numbers on this site mean, how Personal Rating is calculated, and where the data comes from.

Where the data comes from

Player and ship statistics are read live from the official Wargaming Public API every time you search. Battles, win rate, average damage and so on always reflect the latest values the API returns (Wargaming updates a player's totals at the end of each battle).

This is an unofficial, fan-made site. It is not affiliated with the game's publisher.

Personal Rating (PR)

PR is a single number that compares your performance on each ship to the expected performance for that ship. It follows the same idea as the well-known WoWS Numbers / Wahzehd formula. It is not an official in-game stat — it exists only to compare players here.

The formula

For every ship you have played we take three ratios — your average damage, frags and win rate divided by the expected values — then normalise and combine them:

ratios: rDmg, rFrags, rWins (yours ÷ expected)
nDmg = max(0, (rDmg − 0.4) / 0.6)
nFrags = max(0, (rFrags − 0.1) / 0.9)
nWins = max(0, (rWins − 0.7) / 0.3)
PR = 700·nDmg + 300·nFrags + 150·nWins

That weighting is roughly 60% damage, 25% frags, 15% win rate. If you match the expected values exactly on all three, your PR is about 1150 — the middle of the "Average" band.

Where the expected values come from

This is the part that makes PR work without an official source. We use two layers:

  • Indexed averages — every time a player is searched, their per-ship stats are saved. The expected value for a ship is the average across all indexed players who own it. The more the site is used, the better these become.
  • Tier + class fallback — for ships the database has not seen enough yet, a reasonable estimate based on the ship's tier and class is used, so PR works on every ship from day one.
Because of this, PR is only as accurate as the data indexed so far. On rarely-seen ships it leans on the estimate; on popular ships it reflects real indexed averages. It improves automatically over time.

Rankings

The leaderboards are built from the players this site has indexed — not from every player in the game (the API does not expose that). They start small and grow as more players and clans are searched. Opening a clan indexes all its members at once, which is the fastest way to fill the rankings.

  • Player leaderboard: needs at least 100 random battles to appear.
  • Per-ship leaderboard: needs at least 50 battles on that ship.
  • You can search any player's position with the "Find a player" box on each leaderboard.

Other features

  • Compare — put two players side by side, overall and on a shared ship.
  • Clans — search a clan, see its roster with roles, and index its members.
  • Languages — the interface is available in English, Italian, French, German and Spanish.
  • Light / dark theme — follows your system by default; switch any time with the ☀️/🌙 button.
